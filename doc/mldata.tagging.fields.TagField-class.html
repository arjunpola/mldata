Class TagField

source code

A "special" character field that actually works as a relationship to tags "under the hood". This exposes a space-separated string of tags, but does the splitting/reordering/etc. under the hood.

__get__(self, instance, owner=None)

source code 

Tag getter. Returns an instance's tags if accessed on an instance, and all of a model's tags if called on a class. That is, this model:

  class Link(models.Model):
      ...
      tags = TagField()

Lets you do both of these:

  >>> l = Link.objects.get(...)
  >>> l.tags
  'tag1 tag2 tag3'

  >>> Link.tags
  'tag1 tag2 tag3 tag4'
