Class Forum

source code

Very basic outline for a Forum, or group of threads.

The threads and posts fields are updated by the save() methods of their respective models and are used for display purposes.

All of the parent/child recursion code here is borrowed directly from the Satchmo project: http://www.satchmoproject.com/
