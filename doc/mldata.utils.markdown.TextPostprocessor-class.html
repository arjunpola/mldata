Class TextPostprocessor

source code

TextPostprocessors are run after the dom it converted back into text.

Each TextPostprocessor implements a "run" method that takes a pointer to a text string, modifies it as necessary and returns a text string.

TextPostprocessors must extend markdown.TextPostprocessor.
